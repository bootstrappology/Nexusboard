# NexusBoard

[![Live demo](https://img.shields.io/badge/demo-live-blue)](https://bootstrappology.github.io/Nexusboard/)

NexusBoard is a local-first visual idea-linking web app for sketching connections, organizing thoughts, and building concept boards in the browser.

## Features

- Draggable idea nodes with visual connections
- Persistent localStorage board state
- Shareable board URLs with encoded layouts
- Three starter board templates for quick setup
- JSON export / import for backups and sharing
- Undo / redo history with 50 steps
- Inline node editing with double-click support
- PNG export for sharing snapshots

## How to Use

1. Open `index.html` in your browser or visit the live demo.
2. Add nodes and connect them visually to map ideas.
3. Drag nodes to reposition them and edit labels inline.
4. Use the board toolbar to export JSON, import boards, and save snapshots.
5. Share the current board by copying the generated URL.

## Keyboard Shortcuts

- `Ctrl + Z` / `Cmd + Z` — Undo
- `Ctrl + Y` / `Cmd + Y` — Redo
- `Double click` — Edit node label
- `Right click` — Create a new node at cursor
